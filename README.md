# 登校タイマー

子供が小学校に行くまでの残り時間を可視化するブラウザアプリ。

## 主な機能

### 時間表示
- **現在時刻と出発時刻**: 一目で分かる色分け表示（青紫と赤）
- **カウントダウン**: 出発時刻までの残り時間をMM:SS形式で表示
- **時間切れアラーム**: 残り時間が0になると音で通知

### 残り時間バー
- 全体の残り時間を視覚的に表示
- 左から右に向かって減っていく紫のグラデーションバー
- バー内に残り時間を数値表示
- 分母は「やること合計時間 + バッファー時間」で固定

### やることバー
- 各タスクを色分けされたセグメントで表示
- 幅は所要時間に比例
- クリックで完了/未完了をトグル
- 完了したタスクはグレーアウトして左側に移動
- 左端のバッファーセグメントに進捗状況を表示
  - 未着手: 「がんばろう！」
  - 一部完了: 「あとXつ！」
  - 全完了: 「よくできました！」
  - 時間不足: 「⚠️ XX:XX 足りない！」

### レイアウト
- タブレット横置き表示に最適化
- レスポンシブデザイン対応
- 2つのバーが画面の縦方向を等分して表示

## 設定方法

### config.json
```json
{
  "targetTime": "08:00",        // 出発時刻 (HH:MM形式)
  "bufferMinutes": 5,            // 余裕時間（分）
  "tasks": [
    {
      "name": "薬飲む",
      "duration": 3,             // 所要時間（分）
      "color": "#FF6B6B"         // 表示色
    }
    // ...
  ],
  "alarmSound": "alarm.mp3"     // アラーム音ファイル
}
```

### URLパラメータ
`?time=HHMM` で出発時刻を上書きできます。

**例:**
- `index.html?time=0800` → 8:00に出発
- `index.html?time=0730` → 7:30に出発
- `index.html?time=1545` → 15:45に出発

## 技術スタック
- Vanilla JavaScript (TypeScript型チェック有効)
- CSS3 (Flexboxレイアウト)
- フロントエンドのみ (サーバー不要)

## デプロイ
- GitHub Pagesでホスティング可能
- 状態はページリロード時にリセット
