# 登校タイマー
ブラウザで使えるタイマーアプリ。
- あらかじめ指定した時刻までの残り秒数をカウントダウン表示
- 残り秒数が0になるとアラーム音を鳴らす
- 全体カウントダウンとは別に、個別に「やること」が登録できる。
  - 「やること」ごとに所要時間を設定できる。
  - カウントダウン中に、やることの終了ができる。
  - 残り「やること」合計時間よりも全体残り時間が短くなった場合、それが分かる表示にする。
- 残り時間はシンプルなバー表示。「やること」ごとの色分け + 全体残り時間が減っていくイメージ
- 設定はとりあえず隣に置いたjsonファイルで管理
- フロントオンリー: 「やること」の消化状態は保存しない
- タブレット横置きしたときにぴったり表示される配置
- github.ioでホスティング
- vanilla js + cssで実装

## URLパラメータ

### 出発時刻の指定
`?time=HHMM` で出発時刻を上書きできます。

**例:**
- `index.html?time=0800` → 8:00に出発
- `index.html?time=0730` → 7:30に出発
- `index.html?time=1545` → 15:45に出発
